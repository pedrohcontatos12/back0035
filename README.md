Este projeto consiste no desenvolvimento de uma API REST para gerenciamento completo de uma concessionária. A aplicação permite adicionar, remover, editar e consultar informações relacionadas a clientes, carros, vendedores e vendas, oferecendo um CRUD estruturado e funcional para cada entidade.

O sistema foi construído utilizando Java e Spring Boot, com foco em boas práticas e organização do código. Para o banco de dados, foram utilizados MySQL como base principal e H2 Database para testes e ambiente de desenvolvimento. A aplicação inclui a implementação de relacionamentos entre entidades utilizando JPA, além de possuir uma arquitetura bem estruturada dividida em Entities, Repository, Service e Controller.

Este projeto foi desenvolvido como parte de um curso extracurricular realizado no Firjan Senai, visando o aprimoramento das habilidades práticas no desenvolvimento de APIs com Java, Spring Boot e integração com banco de dados.
